Java NIO系列(二) - Buffer

Java编程进阶系列
NIO

前言

在Java NIO中，缓冲区用来临时存储数据，可以理解为是I/O操作中数据暂存的中转站。缓冲区直接为通道(Channel)服务，数据是从通道读入缓冲区，从缓冲区写入到通道中的。

缓冲区本质上是一块可以写入数据，然后可以从中读取数据的内存。这块内存被包装成NIO Buffer对象，并提供了一组方法，用来方便的访问这块内存。
